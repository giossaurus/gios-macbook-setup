# Extensões de navegador

Poucas, mas cada uma com um propósito claro: navegar pelo teclado e reduzir distração.

| Extensão | Por que uso |
|----------|-------------|
| **[Vimium](https://chromewebstore.google.com/detail/vimium/dbepggeogbaibhgnhhndojpepiihcmeb)** | Navegar a web **pelo teclado** (estilo Vim): `f` abre links por letra, `j/k` rola, `/` busca, `H/L` histórico. Combina com o resto do setup keyboard-first. |
| **[Unhook](https://chromewebstore.google.com/detail/unhook-remove-youtube-rec/khncfooichmfjbepaaaebmommgaepoid)** | Remove recomendações, Shorts, comentários e sidebar do YouTube - vê o que veio ver e sai. |
| **[Freedom](https://chromewebstore.google.com/detail/freedom-website-blocker-f/abdkjmofmjelgafcdffaimhgdgpagmop)** | Bloqueia sites que distraem em blocos de foco. |

## Vimium - atalhos que mais uso

| Tecla | Ação |
|-------|------|
| `f` | Mostra "hints" - digite as letras pra clicar um link |
| `j` / `k` | Rolar pra baixo / cima |
| `gg` / `G` | Topo / fim da página |
| `H` / `L` | Voltar / avançar no histórico |
| `/` | Buscar na página |
| `o` | Abrir URL / busca (tipo omnibar) |
| `x` | Fechar aba · `X` reabre a última fechada |

> Configuração do Vimium é feita na própria página de opções da extensão (não há dotfile - mas dá
> pra exportar/importar o JSON de config lá, caso queira versionar depois).
